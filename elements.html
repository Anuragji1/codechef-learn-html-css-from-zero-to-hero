Understanding HTML Elements

HTML elements are the building blocks of web pages. They define the structure and content of the page.

Anatomy of an HTML Element

Here's a breakdown of the key parts of an HTML element:

  • Tags: Enclosed in angle brackets (<>), tags mark the beginning and end of an element.
  • Opening tag: The first tag, containing the element name and any attributes.
  • Content: The text or other elements that appear between the opening and closing tags.
  • Closing tag: The second tag, containing a forward slash (/) before the element name.

Example

Here's an example of an HTML element:

<h1>My name is Anthony Gonsalves</h1>

In this example:

  • The tags are <h1> and </h1>.
  • The opening tag is <h1>.
  • The content is "My name is Anthony Gonsalves".
  • The closing tag is </h1>.
  • The entire unit together is called an HTML element.

This element creates a heading on the web page with the text "My name is Anthony Gonsalves".
